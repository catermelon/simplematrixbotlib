## Security Policy

### Supported Versions

| simplematrixbotlib | Supported          |
|--------------------| ------------------ |
| \>= 3.0            | ✅ |
| < 3.0              | ❌                |

| Python  | Supported          |
|---------| ------------------ |
| \>= 3.9 | ✅ |
| < 3.9   | ❌                |

### Reporting a Vulnerability

If matrix-nio is responsible for the vulnurability, please report it to them via their preferred channels. Otherwise, please send a private message to [@imbev:matrix.org](https://matrix.to/#/@imbev:matrix.org) via Matrix.
